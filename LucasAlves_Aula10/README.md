## 2. Elementos Fundamentais da Rede
Abaixo, o esquema ilustra os quatro componentes essenciais para que ocorra a comunicação em qualquer sistema de rede:

*   **Emissor:** O dispositivo que origina a mensagem (ex: Smartphone).
*   **Receptor:** O destino final da informação (ex: Servidor Web).
*   **Meio de Transmissão:** O caminho físico ou sem fio (ex: Fibra óptica ou Wi-Fi).
*   **Protocolo:** O conjunto de regras/idioma que permite o entendimento (ex: TCP/IP).

## 3. Classificação de Redes por Abrangência
As redes são classificadas de acordo com a área geográfica que cobrem. Confira o quadro comparativo abaixo:

| Tipo | Sigla | Abrangência | Exemplo Real |
| :--- | :--- | :--- | :--- |
| **Personal Area Network** | **PAN** | Pessoal (metros) | Bluetooth entre celular e fone de ouvido. |
| **Local Area Network** | **LAN** | Local (prédio/casa) | Rede Wi-Fi doméstica ou laboratório do IMD. |
| **Metropolitan Area Network** | **MAN** | Cidade | Rede de uma prefeitura ou campus da UFRN. |
| **Wide Area Network** | **WAN** | Países/Continentes | A Internet. |
